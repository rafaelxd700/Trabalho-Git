# Disciplinas do curso de Engenharia de Software

## Fundamentos para Engenharia de Software

A disciplina Fundamentos de Engenharia de Software *introduz os princípios, métodos e técnicas para o desenvolvimento sistemático de softwares de alta qualidade, indo além da simples programação*. Ela aborda o ciclo de vida do software, incluindo análise de requisitos, design, testes, manutenção e gestão de projetos.

## Programação e Sistemas Computacionais I

A disciplina de Programação e Sistemas Computacionais *ensina a base da computação, conectando o desenvolvimento de software (código) ao funcionamento do hardware (componentes físicos) e sistemas operacionais*. Ela capacita alunos a criar algoritmos, entender a arquitetura de computadores, manipular dados binários e otimizar programas, focando no que ocorre "dentro do computador" durante a execução de tarefas.

## Lógica Computacional

A disciplina de Lógica Computacional *é a área da computação que aplica princípios da lógica matemática para resolver problemas, estruturar algoritmos e verificar a corretude de sistemas de software/hardware*. Ela ensina a organizar o raciocínio através de proposições (verdadeiro/falso), conectores lógicos, tabelas verdade e lógica de predicados.

## Criatividade e Inovação

A disciplina de Criatividade e Inovação em Engenharia de Software *capacita alunos a gerar novas ideias (criatividade) e implementá-las com valor (inovação) para resolver problemas reais de software*. Ela utiliza métodos como Design Thinking, Lean Startup e técnicas de ideação para criar soluções tecnológicas originais, melhorando processos e produtos.
